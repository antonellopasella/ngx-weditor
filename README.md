# Ngx Weditor

> A Simple WYSIWYG Editor for Angular6+ Applications.

[![NPM Version][npm-image]][npm-url]


## Install

```bash
npm i -S ngx-weditor
```

## Usage

```javascript
import { NgxWeditorModule } from 'ngx-weditor';

@NgModule({
  imports: [
    NgxWeditorModule
  ]
});
```
